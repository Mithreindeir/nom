# MathParser
A simple program that parses an arbitrary string and simplifies the expression using order of operations.


Example:
input:
7 + 3 * (10 / (12 / (3 + 1) - 1)) / (2 + 3) - 5 - 3 + (8)
output:
7 + 3 * (10 / (12 / (3 + 1) - 1)) / (2 + 3) - 5 - 3 + (8)=10

Input string is a global at the top.
